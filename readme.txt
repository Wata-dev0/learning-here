SQL: Bancos de dados relacionais (como PostgreSQL, MySQL, SQLite, etc.)
NoSQL: Bancos de dados não relacionais (como MongoDB, Firebase, etc.)

outras funcionalidades:

Autenticação e autorização (JWT, OAuth)
Validação de dados (express-validator, Joi, Mongoose validation)
Paginação
Busca e filtros
Soft delete
Upload de arquivos
Envio de e-mails
Cache (Redis)
Auditoria (logs, histórico de ações)
